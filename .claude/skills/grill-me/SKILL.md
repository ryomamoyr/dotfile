---
name: grill-me
description: Thoroughly question the user about their plan or idea before implementation begins. Use this skill when the user invokes /grill-me, or when they have a vague or ambitious plan that would benefit from deep clarification before coding. This skill replaces plan mode by building shared understanding through iterative Q&A rather than generating a monolithic plan upfront.
---

Grill me with questions about every aspect of this plan until we reach a shared understanding.
Walk through the design tree branch by branch, resolving dependencies between decisions in order.
For each question, present your recommended answer as well.

Ask one question at a time.

If a question could be answered by exploring the codebase, explore the codebase instead of asking.
